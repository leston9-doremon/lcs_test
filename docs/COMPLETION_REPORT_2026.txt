📁 DATA STRUCTURE

Your data is stored in: ./data/

   news.json
   ├─ id, title, category, date, excerpt, content, image
   └─ Access via: loadNewsData()

   faculty.json
   ├─ id, name, subject, qualification, experience
   └─ Access via: loadFacultyData()

   management.json
   ├─ id, name, role, contact
   └─ Access via: loadManagementData()

   documents.json
   ├─ id, title, category, date, url
   └─ Access via: loadDocumentsData()

   testimonials.json
   ├─ id, name, role, text, rating
   └─ Access via: loadTestimonialsData()

   contact.json
   ├─ address, phone1, phone2, email, hours
   └─ Access via: loadContactData()


💻 USAGE EXAMPLES
═════════════════════════════════════════════════════════════════════════════════

Example 1: Display News on Homepage
───────────────────────────────────────
<script src="shared-data-loader.js"></script>
<script>
  async function showNews() {
    const news = await loadNewsData();
    news.forEach(article => {
      console.log(article.title);
    });
  }
  document.addEventListener('DOMContentLoaded', showNews);
</script>


Example 2: Display Faculty
───────────────────────────────────────
<script src="shared-data-loader.js"></script>
<script>
  loadFacultyData().then(faculty => {
    faculty.forEach(member => {
      console.log(member.name, member.subject);
    });
  });
</script>


Example 3: Load Everything
───────────────────────────────────────
<script src="shared-data-loader.js"></script>
<script>
  loadAllData().then(data => {
    console.log('News:', data.news);
    console.log('Faculty:', data.faculty);
    console.log('Contact:', data.contact);
  });
</script>


🔄 HOW IT WORKS
═════════════════════════════════════════════════════════════════════════════════

Admin Updates Data:
   1. Opens admin-panel.html
   2. Fills in form (e.g., "Add News Article")
   3. Clicks "Save"
   4. Data → API → JSON file
   5. Toast: "✓ Saved & frontend updated!"

Frontend Loads Data:
   1. Page includes shared-data-loader.js
   2. Calls loadNewsData()
   3. Fetches ./data/news.json
   4. Displays data on page
   5. ✅ Data automatically synced!
