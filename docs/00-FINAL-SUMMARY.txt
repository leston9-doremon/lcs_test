## 🆘 Troubleshooting

**Q: Admin panel won't load?**
```
A: Make sure:
   1. npm start is running
   2. Try: http://localhost:3000/admin-panel.html
   3. Check browser console (F12) for errors
```

**Q: Frontend can't load data?**
```
A: Make sure:
   1. shared-data-loader.js is in root folder
   2. data/ folder exists with JSON files
   3. Server is running
```

**Q: CSS not loading?**
```
A: Make sure:
   1. admin-panel.css is in root folder
   2. admin-panel.html links to it correctly
   3. Browser cache cleared (Cmd+Shift+R)
```
